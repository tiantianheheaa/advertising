# CPM、CPC、CPA、oCPM、oCPC 出价与计费模式全方位对比

广告计费模式本质上是在回答三个问题：

1. **广告主按什么付费**：曝光、点击，还是转化？
2. **平台/媒体按什么优化**：曝光量、点击率，还是转化率？
3. **风险由谁承担**：广告主、媒体，还是广告平台算法？

下面从**定义、原理、预估项、优缺点、适用场景、适合的广告主与媒体**几个角度系统说明。

---

# 一、先区分两个概念：出价模式 vs 计费模式

很多人会把“出价”和“计费”混在一起，但在广告系统中二者并不完全相同。

| 概念 | 含义 | 举例 |
|---|---|---|
| **出价模式** | 广告主告诉系统自己愿意为某种目标付出多少钱 | 愿意为一次转化付 100 元 |
| **计费模式** | 广告平台实际向广告主扣费的事件 | 按曝光扣费、按点击扣费、按转化扣费 |
| **优化目标** | 广告系统实际优化的业务指标 | 点击率、转化率、ROI、GMV |

例如：

- **CPM**：通常是“按曝光出价，按曝光计费”
- **CPC**：通常是“按点击出价，按点击计费”
- **CPA**：通常是“按转化出价，按转化计费”
- **oCPM**：广告主往往按转化目标出价，但平台按曝光计费
- **oCPC**：广告主往往按转化目标出价，但平台按点击计费

这也是 oCPM、oCPC 比传统 CPM、CPC 更复杂的原因。

---

# 二、核心公式：媒体为什么最终都看 eCPM

不管广告主按什么方式付费，媒体最终关心的是：

> **每一千次曝光能赚多少钱，也就是 eCPM。**

不同计费模式都可以换算成 eCPM。

## 1. CPM 模式

```text
eCPM = CPM
```

如果广告主出价 20 元 CPM，那么媒体每千次曝光理论收入就是 20 元。

---

## 2. CPC 模式

```text
eCPM = CPC × CTR × 1000
```

假设：

- CPC = 2 元
- CTR = 2%

则：

```text
eCPM = 2 × 2% × 1000 = 40 元
```

媒体不是只看 CPC 高低，而是看 **CPC × 点击率** 后的千次曝光收益。

---

## 3. CPA 模式

```text
eCPM = CPA × CTR × CVR × 1000
```

假设：

- CPA = 100 元
- CTR = 2%
- CVR = 5%

则：

```text
eCPM = 100 × 2% × 5% × 1000 = 100 元
```

CPA 对媒体来说风险更高，因为要同时依赖点击率和转化率。

---

## 4. oCPM / oCPC 模式

oCPM、oCPC 本质上也是通过模型预估转化价值，再换算成可参与竞价的 eCPM。

常见形式：

```text
预估 eCPM = 目标 CPA × pCTR × pCVR × 1000
```

其中：

- **pCTR**：预估点击率
- **pCVR**：预估转化率
- **目标 CPA**：广告主可接受的一次转化成本

如果是电商广告，也可能使用：

```text
预估 eCPM = 预估 GMV × 佣金率 / 曝光量 × 1000
```

或：

```text
预估 eCPM = pCTR × pCVR × 客单价 × 毛利率 × 1000
```

---

# 三、五种模式总览对比

| 模式 | 广告主付费事件 | 平台优化目标 | 风险主要承担方 | 对模型要求 | 典型目标 |
|---|---|---|---|---|---|
| **CPM** | 曝光 | 曝光量、覆盖、人群触达 | 广告主 | 较低 | 品牌曝光 |
| **CPC** | 点击 | 点击率 | 广告主和媒体共同承担 | 中等 | 引流、访问 |
| **CPA** | 转化 | 转化量 | 媒体/平台承担更多 | 较高 | 注册、下单、激活 |
| **oCPM** | 曝光 | 转化、ROI、GMV | 广告主承担扣费风险，平台承担优化责任 | 很高 | 效果广告规模化 |
| **oCPC** | 点击 | 转化、ROI、GMV | 广告主和平台共同承担 | 高 | 转化型投放 |

---

# 四、CPM：按千次曝光计费

## 1. 定义

**CPM，Cost Per Mille**，即每千次曝光成本。

广告主为广告展示次数付费，通常以一千次曝光为计费单位。

```text
广告费用 = 曝光次数 / 1000 × CPM 出价
```

例如：

- CPM = 30 元
- 曝光 100 万次

则费用为：

```text
1000000 / 1000 × 30 = 30000 元
```

---

## 2. 原理

CPM 模式下，广告系统主要关注：

- 广告是否能展示给目标用户
- 是否符合定向条件
- 是否满足频控
- 是否在预算范围内
- 是否能最大化媒体曝光收益

广告主买的是**展示机会**，不保证点击，也不保证转化。

---

## 3. 广告主需要预估什么

| 预估项 | 说明 |
|---|---|
| **目标人群规模** | 需要覆盖多少用户 |
| **有效曝光频次** | 每个用户平均看几次广告合适 |
| **预期 CTR** | 曝光后可能有多少点击 |
| **预期 CVR** | 点击后可能有多少转化 |
| **品牌曝光价值** | 一次曝光对品牌认知、心智、搜索提升的价值 |
| **可接受 CPM** | 每千次曝光最多愿意付多少钱 |
| **预算消耗速度** | 是否需要集中爆发，还是长期平稳触达 |

---

## 4. 媒体需要预估什么

| 预估项 | 说明 |
|---|---|
| **广告位曝光库存** | 可以提供多少曝光量 |
| **广告位质量** | 首屏、信息流、开屏、视频贴片等价值不同 |
| **用户画像价值** | 高消费、高转化人群通常 CPM 更高 |
| **填充率** | 广告请求能否被广告填满 |
| **售卖底价** | 每千次曝光最低多少钱可接受 |
| **用户体验影响** | 广告密度过高可能伤害留存 |

---

## 5. 优点

### 对广告主

- **适合大规模品牌曝光**
- 可以快速覆盖大量用户
- 预算和曝光量容易规划
- 适合新品发布、大促预热、品牌心智建设

### 对媒体

- 收入稳定、可预测
- 不依赖点击和转化
- 适合优质曝光资源售卖
- 对归因和转化链路依赖较低

---

## 6. 缺点

### 对广告主

- 只保证曝光，不保证点击和转化
- 如果人群定向不准，浪费较大
- 容易出现“看到了但没效果”的问题
- 对品牌效果评估要求较高

### 对媒体

- 如果广告主更关注效果，CPM 售卖可能不够有吸引力
- 需要证明流量质量和人群价值
- 对可见曝光、无效曝光、反作弊要求较高

---

## 7. 适用场景

| 场景 | 是否适合 CPM |
|---|---|
| 品牌广告 | 非常适合 |
| 新品发布 | 适合 |
| 大促预热 | 适合 |
| 心智占领 | 适合 |
| 精准转化 | 不一定适合 |
| 中小广告主直接获客 | 通常不优先 |

---

## 8. 适合什么广告主

**适合：**

- 品牌广告主
- 大型消费品公司
- 汽车、手机、奢侈品、快消品牌
- 有较强预算和品牌建设诉求的广告主
- 对曝光、覆盖、频次、品牌认知有明确目标的广告主

**不太适合：**

- 预算很小的中小商家
- 只看 ROI 的效果广告主
- 转化链路短、强考核成本的广告主

---

## 9. 适合什么媒体

**适合：**

- 大流量媒体
- 品牌安全性强的媒体
- 用户质量高的平台
- 首页、开屏、焦点图、视频贴片等强曝光资源
- 内容调性清晰、品牌愿意投放的媒体

**不太适合：**

- 流量质量不稳定的媒体
- 点击或转化能力弱且无法证明曝光价值的媒体
- 无法提供可见性和反作弊能力的媒体

---

# 五、CPC：按点击计费

## 1. 定义

**CPC，Cost Per Click**，即每次点击成本。

广告主只有在用户点击广告时才付费。

```text
广告费用 = 点击次数 × CPC 出价
```

例如：

- CPC = 3 元
- 点击 10000 次

费用为：

```text
3 × 10000 = 30000 元
```

---

## 2. 原理

CPC 模式下，广告系统会更关注：

- 哪些广告更容易被点击
- 哪些用户更可能点击
- 哪些广告位点击率更高
- 如何在点击收益和用户体验之间平衡

媒体的收益可以表示为：

```text
媒体 eCPM = CPC × CTR × 1000
```

因此媒体并不只喜欢高 CPC 广告，也喜欢高 CTR 广告。

---

## 3. 广告主需要预估什么

| 预估项 | 说明 |
|---|---|
| **可接受点击成本** | 一次访问最多值多少钱 |
| **点击后转化率** | 点击进入落地页后，有多少会注册、购买、留资 |
| **落地页承接能力** | 页面加载速度、转化路径、商品吸引力 |
| **单次转化价值** | 每个注册、订单、线索的商业价值 |
| **预期 ROI** | 点击成本是否能被后续收益覆盖 |
| **无效点击比例** | 是否存在误点、刷点、低质量点击 |

---

## 4. 媒体需要预估什么

| 预估项 | 说明 |
|---|---|
| **CTR** | 广告展示后被点击的概率 |
| **广告样式点击效率** | 信息流、搜索、推荐位、Banner 的点击率差异 |
| **误点击风险** | 误点过高会损害广告主和平台信任 |
| **点击质量** | 点击是否真实、是否有后续行为 |
| **广告相关性** | 广告和内容、人群是否匹配 |
| **eCPM 收益** | CPC × CTR × 1000 后是否高于其他广告 |

---

## 5. 优点

### 对广告主

- 只为点击付费，比 CPM 更接近效果
- 适合拉新、引流、访问落地页
- 数据反馈比品牌曝光更直接
- 可通过落地页优化提升转化

### 对媒体

- 点击率高的广告位可以获得更高收益
- 适合搜索广告、信息流广告等强意图场景
- 比 CPA 对媒体风险更低

---

## 6. 缺点

### 对广告主

- 点击不等于转化
- 容易遇到低质量点击、误点击、无效点击
- 如果落地页转化差，CPC 也可能亏损
- 对创意标题党有一定激励，可能带来无效流量

### 对媒体

- 收入依赖 CTR，不如 CPM 稳定
- 如果广告与用户不匹配，收益较低
- 需要较强反作弊能力
- 过度追求点击可能伤害用户体验

---

## 7. 适用场景

| 场景 | 是否适合 CPC |
|---|---|
| 搜索广告 | 非常适合 |
| 电商商品引流 | 适合 |
| 内容落地页访问 | 适合 |
| App 下载页访问 | 适合 |
| 线索收集前链路 | 适合 |
| 品牌纯曝光 | 不如 CPM |
| 深度转化优化 | 不如 oCPM/oCPC |

---

## 8. 适合什么广告主

**适合：**

- 希望获得站外/站内访问流量的广告主
- 电商商家
- 搜索广告主
- 内容营销广告主
- 有较好落地页承接能力的广告主
- 能够追踪点击后转化的广告主

**不太适合：**

- 完全没有转化追踪能力的广告主
- 落地页体验差的广告主
- 只关注最终成交成本的广告主

---

## 9. 适合什么媒体

**适合：**

- 搜索引擎
- 电商平台
- 信息流媒体
- 内容平台
- 点击意图明确的垂直媒体

**不太适合：**

- 用户点击意愿很低的纯展示媒体
- 广告样式弱、互动弱的媒体
- 误点击率高、流量质量难验证的媒体

---

# 六、CPA：按转化计费

## 1. 定义

**CPA，Cost Per Action**，即按特定行为计费。

这里的 Action 可以是：

- 注册
- 下载
- 激活
- 留资
- 下单
- 支付
- 首购
- 预约
- 授信
- 开户

```text
广告费用 = 转化次数 × CPA 出价
```

例如：

- CPA = 80 元
- 有效注册 1000 个

费用为：

```text
80 × 1000 = 80000 元
```

---

## 2. 原理

CPA 模式下，广告主只为最终行为付费。

媒体或广告平台需要承担更多风险，因为：

- 曝光了不一定点击
- 点击了不一定转化
- 转化数据可能延迟
- 转化归因可能有争议
- 广告主落地页、商品价格、库存、客服都会影响转化

媒体侧需要把 CPA 转换成预估 eCPM：

```text
预估 eCPM = CPA × pCTR × pCVR × 1000
```

---

## 3. 广告主需要预估什么

| 预估项 | 说明 |
|---|---|
| **单次转化价值** | 一个注册、订单、线索、激活值多少钱 |
| **可接受 CPA** | 每获得一次转化最多花多少钱 |
| **后端变现能力** | 用户生命周期价值、复购、留存、客单价 |
| **转化定义** | 什么行为算有效转化 |
| **转化质量标准** | 是否要求真实用户、有效线索、去重、反作弊 |
| **归因窗口** | 点击后 1 天、7 天、30 天内转化是否计入 |
| **数据回传能力** | 是否能及时准确回传转化数据 |

---

## 4. 媒体需要预估什么

| 预估项 | 说明 |
|---|---|
| **pCTR** | 用户点击概率 |
| **pCVR** | 点击后转化概率 |
| **转化延迟** | 用户从点击到转化需要多久 |
| **归因准确性** | 转化是否能被正确归因到媒体 |
| **广告主履约能力** | 落地页、商品、库存、客服是否影响转化 |
| **作弊风险** | 虚假注册、虚假激活、羊毛党 |
| **预估 eCPM** | CPA × pCTR × pCVR × 1000 |

---

## 5. 优点

### 对广告主

- 风险较低，只为结果付费
- 成本可控，直接对应业务目标
- 适合明确转化目标的投放
- 便于 ROI 测算

### 对媒体

- 如果流量转化质量高，可以获得较高收益
- 有助于证明媒体效果价值
- 适合强效果联盟、垂直流量、私域流量合作

---

## 6. 缺点

### 对广告主

- 单价通常较高
- 优质媒体不一定愿意接受纯 CPA
- 可能出现低质量转化、虚假转化
- 如果 CPA 设置过低，拿不到量
- 需要强转化追踪和反作弊能力

### 对媒体

- 风险较大，曝光和点击可能无法变现
- 收入受广告主落地页、商品、价格、客服影响
- 归因争议多
- 回款和结算周期可能较长
- 对数据透明度要求高

---

## 7. 适用场景

| 场景 | 是否适合 CPA |
|---|---|
| App 激活 | 适合 |
| 注册拉新 | 适合 |
| 金融留资 | 适合 |
| 教育线索 | 适合 |
| 游戏注册/付费 | 适合 |
| 电商 CPS/下单 | 适合，但需风控 |
| 品牌曝光 | 不适合 |
| 高客单复杂决策 | 谨慎适用 |

---

## 8. 适合什么广告主

**适合：**

- 有明确转化目标的广告主
- App、游戏、教育、金融、电商广告主
- 后端转化价值清晰的广告主
- 有转化数据回传和反作弊能力的广告主
- 可以定义清楚“有效转化”的广告主

**不太适合：**

- 转化周期非常长的广告主
- 转化难以准确归因的广告主
- 品牌心智类广告主
- 数据回传能力弱的广告主

---

## 9. 适合什么媒体

**适合：**

- 垂直媒体
- 联盟媒体
- App 分发渠道
- 内容种草平台
- 高意向搜索或导购媒体
- 能提供高转化人群的媒体

**不太适合：**

- 纯曝光媒体
- 转化链路不可控的媒体
- 无法获得转化回传的媒体
- 品牌型优质流量资源，因为它们更愿意卖 CPM 或 CPC

---

# 七、oCPM：优化型 CPM

## 1. 定义

**oCPM，Optimized Cost Per Mille**，即优化型千次曝光计费。

它的特点是：

> **广告主通常设置转化目标或目标 CPA，平台按曝光计费，但系统按照转化效果进行优化。**

也就是说：

- **计费方式**：按曝光计费
- **优化目标**：转化、下单、激活、ROI、GMV
- **出价表达**：广告主可能填写目标 CPA、目标 ROI 或转化价值

---

## 2. 原理

oCPM 的核心是广告平台通过模型预估每一次曝光的转化价值。

常见排序公式：

```text
预估 eCPM = 目标 CPA × pCTR × pCVR × 1000
```

如果广告主目标 CPA 是 100 元：

- 某用户看到广告后点击概率 pCTR = 3%
- 点击后转化概率 pCVR = 5%

则：

```text
预估 eCPM = 100 × 3% × 5% × 1000 = 150 元
```

广告系统会认为这次曝光价值较高，可以参与更高强度的竞价。

---

## 3. 广告主需要预估什么

| 预估项 | 说明 |
|---|---|
| **目标 CPA** | 愿意为一次转化支付多少钱 |
| **转化价值** | 一次转化带来的收入、毛利、LTV |
| **预算规模** | oCPM 需要足够预算帮助模型学习 |
| **转化事件选择** | 选择浅层转化还是深层转化 |
| **数据回传质量** | 转化数据是否及时、准确、稳定 |
| **冷启动成本** | 学习期可能成本波动较大 |
| **ROI 目标** | 尤其适用于电商、本地生活、游戏等场景 |

---

## 4. 媒体/平台需要预估什么

oCPM 对媒体和平台模型要求很高。

| 预估项 | 说明 |
|---|---|
| **pCTR** | 用户看到广告后点击概率 |
| **pCVR** | 点击后转化概率 |
| **曝光价值** | 每一次曝光可能带来的转化收益 |
| **用户长期价值** | 部分场景需要预估 LTV |
| **转化延迟** | 下单、付费、复购可能滞后 |
| **预算消耗速度** | 防止过快消耗或跑不出去 |
| **模型校准** | 预估值要和真实转化率接近 |
| **探索与利用平衡** | 既要找新流量，也要稳定拿高转化流量 |

---

## 5. 优点

### 对广告主

- 相比传统 CPM，更关注转化结果
- 可以规模化拿量
- 不需要广告主逐个媒体手动调价
- 适合复杂人群定向和智能投放
- 在模型成熟时，效果通常优于手动 CPC/CPM

### 对媒体/平台

- 可以把转化率高的流量卖出更高价格
- 提升整体 eCPM 和流量变现效率
- 更容易连接品牌预算和效果预算
- 有利于平台算法形成壁垒

---

## 6. 缺点

### 对广告主

- 虽然目标是转化，但仍然**按曝光扣费**
- 学习期成本可能波动
- 如果模型不准，可能花了钱但转化不达标
- 对转化数据回传依赖强
- 出价过低可能没有量，出价过高可能成本失控
- 可解释性弱，广告主不一定知道钱花到了哪里

### 对媒体/平台

- 模型建设复杂
- 需要大量训练数据
- 冷启动难
- 转化延迟会影响优化
- 需要处理归因、反作弊、预算、频控等复杂问题
- 如果效果不稳定，广告主信任会下降

---

## 7. 适用场景

| 场景 | 是否适合 oCPM |
|---|---|
| 信息流效果广告 | 非常适合 |
| App 拉新 | 适合 |
| 电商下单 | 适合 |
| 直播间引流成交 | 适合 |
| 游戏激活/付费 | 适合 |
| 本地生活转化 | 适合 |
| 品牌纯曝光 | 不一定 |
| 小预算冷启动 | 谨慎适用 |

---

## 8. 适合什么广告主

**适合：**

- 有稳定转化数据的广告主
- 日预算较充足的广告主
- 电商、游戏、App、本地生活、教育、金融等效果广告主
- 有明确 CPA、ROI、GMV 目标的广告主
- 接受算法自动优化的广告主

**不太适合：**

- 预算极小的广告主
- 转化数据很少的广告主
- 无法回传转化事件的广告主
- 转化周期特别长、样本稀疏的广告主
- 强依赖人工控制投放位置的广告主

---

## 9. 适合什么媒体

**适合：**

- 大流量信息流平台
- 短视频平台
- 电商平台
- 搜索与推荐融合平台
- 用户数据丰富的平台
- 能掌握曝光、点击、转化闭环数据的平台

**不太适合：**

- 小流量媒体
- 数据闭环弱的媒体
- 无法获得转化回传的媒体
- 模型能力较弱的广告网络

# 八、oCPC：优化型 CPC

## 1. 定义

**oCPC，Optimized Cost Per Click**，即优化型点击计费。

它的特点是：

> **广告主通常设置目标转化成本，平台按照转化目标优化流量，但实际按点击计费。**

也就是说：

- **计费方式**：按点击计费
- **优化目标**：转化、下单、注册、ROI
- **出价表达**：广告主可能填写目标 CPA 或转化目标

---

## 2. 原理

oCPC 是介于 CPC 和 oCPM 之间的一种模式。

传统 CPC 只看点击：

```text
收益 = CPC × 点击数
```

oCPC 不仅看点击，还会判断点击后的转化概率：

```text
预估转化价值 = 目标 CPA × pCVR
```

平台再结合 pCTR 进行排序和流量分配。

常见竞价逻辑可理解为：

```text
预估 eCPM = 目标 CPA × pCTR × pCVR × 1000
```

但实际扣费时，仍然通常按点击发生扣费。

---

## 3. 广告主需要预估什么

| 预估项 | 说明 |
|---|---|
| **目标 CPA** | 一次转化可接受成本 |
| **转化事件** | 注册、表单、下单、激活等 |
| **点击价值** | 点击后转化概率和商业价值 |
| **落地页承接能力** | 页面质量直接影响 CVR |
| **历史转化数据** | oCPC 通常需要一定转化样本 |
| **学习期预算** | 需要给系统探索空间 |
| **转化回传稳定性** | 影响模型优化效果 |

---

## 4. 媒体/平台需要预估什么

| 预估项 | 说明 |
|---|---|
| **pCTR** | 曝光到点击的概率 |
| **pCVR** | 点击到转化的概率 |
| **点击质量** | 哪些点击更可能转化 |
| **无效点击率** | 过滤低质量点击 |
| **转化归因** | 判断点击是否带来后续转化 |
| **目标成本达成率** | 实际 CPA 是否接近广告主目标 |
| **预算消耗节奏** | 控制放量与成本稳定 |

---

## 5. 优点

### 对广告主

- 比普通 CPC 更接近转化目标
- 仍然按点击付费，理解成本较低
- 能减少低质量点击
- 适合从手动 CPC 逐步过渡到智能投放
- 对中等规模广告主较友好

### 对媒体/平台

- 比 CPA 风险更低，因为点击即可计费
- 比普通 CPC 更能体现流量质量
- 有利于提升广告主转化效果和续投率
- 适合搜索、信息流、效果广告场景

---

## 6. 缺点

### 对广告主

- 点击扣费仍然会发生，不能保证最终转化
- 模型学习期可能不稳定
- 转化数据不足时效果不佳
- 目标 CPA 设置不合理会导致跑量困难或成本超标
- 仍需防范无效点击和误点击

### 对媒体/平台

- 需要比普通 CPC 更强的转化预估能力
- 需要广告主稳定回传转化数据
- 如果转化链路复杂，模型难度较高
- 归因争议仍然存在

---

## 7. 适用场景

| 场景 | 是否适合 oCPC |
|---|---|
| 搜索效果广告 | 非常适合 |
| 信息流线索广告 | 适合 |
| 电商商品推广 | 适合 |
| 教育/金融表单 | 适合 |
| App 下载后激活 | 适合 |
| 品牌纯曝光 | 不适合 |
| 极低转化量场景 | 谨慎适用 |

---

## 8. 适合什么广告主

**适合：**

- 从 CPC 转向转化优化的广告主
- 有一定转化数据积累的广告主
- 教育、金融、家装、本地服务等线索广告主
- 电商中小商家
- 搜索广告主
- 有明确 CPA 目标但还不适合纯 CPA 的广告主

**不太适合：**

- 完全没有转化追踪的广告主
- 转化样本极少的广告主
- 只关注品牌曝光的广告主
- 目标 CPA 设置严重偏离市场价格的广告主

---

## 9. 适合什么媒体

**适合：**

- 搜索媒体
- 信息流平台
- 电商平台
- 垂直内容平台
- 有较强点击与转化数据的媒体

**不太适合：**

- 点击行为少的纯展示广告位
- 无法追踪转化的媒体
- 低质量点击较多的平台

# 九、五种模式详细对比表

## 1. 定义与计费对比

| 模式 | 全称 | 广告主实际为谁付费 | 是否直接保证效果 | 常见优化方向 |
|---|---|---|---|---|
| **CPM** | Cost Per Mille | 曝光 | 否 | 覆盖、曝光、频次 |
| **CPC** | Cost Per Click | 点击 | 否 | 点击率、访问量 |
| **CPA** | Cost Per Action | 转化行为 | 是，相对更接近业务结果 | 转化量、转化成本 |
| **oCPM** | Optimized CPM | 曝光 | 不直接保证，但按转化优化 | 转化、ROI、GMV |
| **oCPC** | Optimized CPC | 点击 | 不直接保证，但按转化优化 | 转化、ROI、GMV |

---

## 2. 风险分配对比

| 模式 | 广告主风险 | 媒体风险 | 平台算法风险 | 说明 |
|---|---:|---:|---:|---|
| **CPM** | 高 | 低 | 低 | 广告主买曝光，点击和转化风险主要由广告主承担 |
| **CPC** | 中 | 中 | 中 | 广告主承担点击后不转化风险，媒体承担无点击风险 |
| **CPA** | 低 | 高 | 高 | 广告主只为转化付费，媒体承担曝光点击不转化风险 |
| **oCPM** | 中高 | 中 | 高 | 广告主按曝光付费，但依赖平台优化转化 |
| **oCPC** | 中 | 中 | 高 | 广告主按点击付费，平台负责筛选高转化点击 |

---

## 3. 对数据和模型要求对比

| 模式 | 是否需要 pCTR | 是否需要 pCVR | 是否强依赖转化回传 | 模型复杂度 |
|---|---|---|---|---|
| **CPM** | 可选 | 不强依赖 | 否 | 低 |
| **CPC** | 需要 | 可选 | 不强依赖 | 中 |
| **CPA** | 需要 | 需要 | 是 | 高 |
| **oCPM** | 强需要 | 强需要 | 是 | 很高 |
| **oCPC** | 需要 | 强需要 | 是 | 高 |

---

## 4. 广告主预算控制对比

| 模式 | 成本可控性 | 跑量能力 | 成本波动 | 说明 |
|---|---|---|---|---|
| **CPM** | 曝光成本可控 | 强 | 低 | 买多少曝光比较清楚 |
| **CPC** | 点击成本可控 | 中高 | 中 | 点击成本可控，但转化成本不一定 |
| **CPA** | 转化成本最可控 | 可能受限 | 低到中 | 成本确定，但可能拿不到量 |
| **oCPM** | 目标成本可控但非绝对 | 强 | 中高 | 放量能力强，学习期波动大 |
| **oCPC** | 中等可控 | 中高 | 中 | 比 CPC 更关注转化，比 oCPM 稳一些 |

---

## 5. 适用营销目标对比

| 营销目标 | 最适合模式 |
|---|---|
| 品牌曝光 | **CPM** |
| 新品发布 | **CPM / oCPM** |
| 网站访问 | **CPC** |
| 搜索引流 | **CPC / oCPC** |
| 注册拉新 | **CPA / oCPC / oCPM** |
| App 激活 | **CPA / oCPM / oCPC** |
| 电商下单 | **oCPM / oCPC / CPA / CPS** |
| 线索收集 | **oCPC / CPA** |
| ROI 优化 | **oCPM / oCPC** |
| 大规模效果放量 | **oCPM** |

# 十、广告主分别需要预估什么

## 1. CPM 下广告主重点预估

广告主主要预估：

- **这批曝光是否覆盖目标人群**
- **每千次曝光值多少钱**
- **曝光后是否能带来品牌认知提升**
- **频次多少合适**
- **是否存在无效曝光**
- **曝光转点击、转化的大致漏斗**

适合广告主内部有品牌研究、市场调研、广告后链路分析能力。

---

## 2. CPC 下广告主重点预估

广告主主要预估：

- **一次点击值多少钱**
- **点击后落地页转化率是多少**
- **每次转化的真实成本是多少**

核心公式：

```text
预估 CPA = CPC / CVR
```

例如：

- CPC = 2 元
- CVR = 4%

则：

```text
预估 CPA = 2 / 4% = 50 元
```

如果广告主能接受的 CPA 是 80 元，这个 CPC 就可能是合理的。

---

## 3. CPA 下广告主重点预估

广告主主要预估：

- **一次有效转化的商业价值**
- **可接受 CPA**
- **转化质量**
- **后端 LTV**
- **作弊和无效转化比例**

核心公式：

```text
可接受 CPA ≤ 单个用户生命周期价值 × 毛利率
```

如果一个新用户生命周期价值是 500 元，毛利率是 30%，那么理论最大获客成本为：

```text
500 × 30% = 150 元
```

---

## 4. oCPM 下广告主重点预估

广告主主要预估：

- **目标 CPA 是否合理**
- **预算是否足够模型学习**
- **转化事件是否太深**
- **转化回传是否及时**
- **是否允许系统探索流量**
- **实际成本是否围绕目标成本波动**

oCPM 不适合只看单次曝光成本，而要看：

```text
实际 CPA = 消耗 / 转化数
```

以及：

```text
实际 ROI = 转化收益 / 广告消耗
```

---

## 5. oCPC 下广告主重点预估

广告主主要预估：

- **目标 CPA**
- **点击后转化率**
- **点击质量**
- **落地页承接能力**
- **转化样本量**
- **实际点击成本与转化成本**

oCPC 的核心判断是：

> 系统买来的点击，是否比普通 CPC 更容易转化。

---

# 十一、媒体分别需要预估什么

## 1. CPM 下媒体重点预估

媒体主要预估：

- 曝光库存
- 广告位可见性
- 用户画像价值
- 品牌安全性
- 底价
- 填充率
- 频控对库存的影响

媒体只要曝光资源强、用户质量高，就适合 CPM 售卖。

---

## 2. CPC 下媒体重点预估

媒体主要预估：

- CTR
- 点击真实性
- 误点击率
- 广告样式点击效率
- 点击后质量
- CPC 转换后的 eCPM

核心判断：

```text
CPC × CTR × 1000 是否高于 CPM 广告收益
```

---

## 3. CPA 下媒体重点预估

媒体主要预估：

- CTR
- CVR
- 转化真实性
- 归因准确性
- 转化延迟
- 广告主落地页质量
- 作弊风险

核心判断：

```text
CPA × CTR × CVR × 1000 是否高于其他广告收益
```

---

## 4. oCPM 下媒体/平台重点预估

媒体或平台主要预估：

- pCTR
- pCVR
- 用户转化倾向
- 广告主目标成本合理性
- 转化回传质量
- 出价与预算消耗关系
- 长期 ROI
- 流量探索价值

oCPM 是对平台模型能力要求最高的模式之一。

---

## 5. oCPC 下媒体/平台重点预估

媒体或平台主要预估：

- pCTR
- pCVR
- 哪些点击更有价值
- 哪些点击是无效点击
- 转化归因是否准确
- 目标 CPA 是否可达成
- 点击成本和转化成本是否稳定

---

# 十二、不同类型广告主如何选择

## 1. 品牌广告主

| 特征 | 推荐模式 |
|---|---|
| 关注曝光、覆盖、频次、品牌认知 | **CPM** |
| 有一定效果诉求，但仍重视曝光 | **CPM + oCPM** |
| 希望兼顾品牌和转化 | **oCPM** |

典型行业：

- 快消
- 汽车
- 手机
- 家电
- 奢侈品
- 大型零售品牌

---

## 2. 电商广告主

| 特征 | 推荐模式 |
|---|---|
| 推广商品详情页 | **CPC / oCPC** |
| 追求下单和 ROI | **oCPM / oCPC** |
| 有成交分佣 | **CPA / CPS** |
| 大促期间放量 | **CPM + oCPM** |

电商广告主最核心看：

```text
ROI = 成交金额 / 广告消耗
```

或：

```text
广告毛利 ROI = 毛利 / 广告消耗
```

---

## 3. App / 游戏广告主

| 特征 | 推荐模式 |
|---|---|
| 下载 | **CPC / CPA** |
| 激活 | **CPA / oCPM / oCPC** |
| 注册 | **CPA / oCPC** |
| 付费用户获取 | **oCPM / oCPC** |
| 长期 LTV 优化 | **oCPM** |

游戏广告主往往不只看激活，还看：

- 次日留存
- 7 日留存
- 付费率
- ARPU
- LTV

---

## 4. 教育、金融、家装、本地服务广告主

| 特征 | 推荐模式 |
|---|---|
| 表单收集 | **oCPC / CPA** |
| 电话咨询 | **CPA / oCPC** |
| 销售线索 | **CPA / oCPC** |
| 高质量线索 | **oCPM / oCPC** |

这类广告主需要特别关注：

- 线索真实性
- 线索接通率
- 后续成交率
- 重复线索
- 羊毛党和虚假表单

---

## 5. 中小广告主

| 特征 | 推荐模式 |
|---|---|
| 预算较小 | **CPC / oCPC** |
| 数据较少 | 先 CPC，再逐步 oCPC |
| 转化追踪弱 | CPC 更稳 |
| 有明确转化目标 | oCPC 或 CPA |

中小广告主不建议一开始就盲目使用深度 oCPM，因为：

- 转化样本少
- 模型学习不足
- 预算不够探索
- 成本波动可能较大

---

# 十三、不同类型媒体如何选择

## 1. 大流量综合平台

适合：

- **CPM**
- **CPC**
- **oCPM**
- **oCPC**

原因：

- 流量大
- 用户数据丰富
- 模型训练样本多
- 可以做复杂定向和转化优化

典型媒体：

- 电商平台
- 短视频平台
- 社交平台
- 搜索平台
- 内容信息流平台

---

## 2. 搜索媒体

最适合：

- **CPC**
- **oCPC**

原因：

- 用户意图强
- 点击价值高
- 关键词可以表达需求
- 点击后转化链路更明确

---

## 3. 信息流媒体

适合：

- **CPM**
- **CPC**
- **oCPM**
- **oCPC**

原因：

- 流量规模大
- 用户停留时间长
- 可以通过模型预测兴趣和转化
- 适合智能投放

---

## 4. 垂直媒体

适合：

- **CPM**
- **CPC**
- **CPA**

原因：

- 用户人群明确
- 行业相关性强
- 广告主愿意为精准人群付费

例如：

- 母婴媒体
- 汽车媒体
- 财经媒体
- 房产媒体
- 医疗健康内容平台

---

## 5. 联盟媒体

适合：

- **CPC**
- **CPA**
- **CPS**

原因：

- 流量来源分散
- 需要按效果结算
- 广告主更关注最终转化
- 需要较强反作弊和归因能力

# 十四、如何选择：一张决策表

| 你的目标 | 优先选择 | 原因 |
|---|---|---|
| 想让更多人看到品牌 | **CPM** | 曝光确定性强 |
| 想获得访问流量 | **CPC** | 只为点击付费 |
| 想获得注册/激活/线索 | **CPA / oCPC** | 更贴近转化 |
| 想大规模获取转化 | **oCPM** | 放量能力强 |
| 想从 CPC 过渡到转化优化 | **oCPC** | 风险和智能化较均衡 |
| 预算小、数据少 | **CPC** | 易理解、易控制 |
| 预算充足、数据闭环好 | **oCPM** | 更适合算法优化 |
| 流量质量强、转化链路闭环 | **oCPM / oCPC** | 可提升整体效率 |
| 媒体强曝光资源 | **CPM** | 曝光价值更容易变现 |
| 媒体强意图流量 | **CPC / oCPC** | 点击和转化价值高 |
| 媒体转化质量强 | **CPA / oCPM / oCPC** | 可以获得更高收益 |

# 十五、五种模式的核心结论

## 1. CPM

**买曝光，适合品牌。**

- 广告主风险较高
- 媒体收益稳定
- 适合品牌曝光、覆盖、频次控制
- 不适合强转化考核的小预算广告主

---

## 2. CPC

**买点击，适合引流。**

- 比 CPM 更接近效果
- 但点击不等于转化
- 适合搜索、电商、内容落地页
- 需要关注点击质量和落地页转化

---

## 3. CPA

**买转化，适合结果导向。**

- 广告主风险最低
- 媒体风险最高
- 对归因、反作弊、转化回传要求高
- 适合注册、激活、留资、下单等明确行为

---

## 4. oCPM

**按曝光计费，但按转化优化，适合大规模效果广告。**

- 放量能力强
- 模型要求最高
- 学习期波动较大
- 适合预算充足、数据闭环完整的广告主和平台

---

## 5. oCPC

**按点击计费，但按转化优化，适合从点击到转化的过渡。**

- 比 CPC 更智能
- 比 oCPM 风险相对低一些
- 适合搜索、线索、电商、App 拉新
- 需要一定转化数据支持

---

# 十六、简化记忆版

| 模式 | 一句话理解 |
|---|---|
| **CPM** | 我买的是曝光，用户点不点、买不买是我的风险 |
| **CPC** | 我买的是点击，点击后转不转化是我的风险 |
| **CPA** | 我买的是结果，没结果基本不付费 |
| **oCPM** | 我按曝光付费，但让系统帮我找更可能转化的人 |
| **oCPC** | 我按点击付费，但让系统帮我筛选更可能转化的点击 |

---

# 十七、最终建议

如果从广告主视角选择：

```text
品牌曝光：CPM
访问引流：CPC
明确转化：CPA
规模化转化：oCPM
稳健转化优化：oCPC
```

如果从媒体视角选择：

```text
强曝光资源：CPM
强点击意图：CPC / oCPC
强转化能力：CPA / oCPM / oCPC
大流量+强模型：oCPM
搜索类媒体：CPC / oCPC
```

**总体来看：**

- **CPM 更偏品牌**
- **CPC 更偏流量**
- **CPA 更偏结果**
- **oCPM 更偏智能化规模转化**
- **oCPC 更偏点击到转化之间的平衡优化**
